Voice Recorder

Description

This microservice takes your uploaded file and returns some metadata about the file.

Submit a file to view its metadata.

Allow this page to access your microphone, then click the mic to start recording. Once you're done, click the mic again to stop recording. The audio output will be saved as a WAV file and uploaded to the file upload question on this page through an AJAX request to our file upload API. (In a live survey, the file upload question would be hidden.)


Works on Chrome, Firefox, Edge, Opera, and the Android browser. Does NOT work on Safari, IE, or iOS mobile devices.
